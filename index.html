About Me

            My story begins In 2019 when I started teaching myself web development with the goal of becoming a front-end web developer. I had programmed in Objective-C for the Mac platform before, but I had no idea what development for the web actually entailed. After getting comfortable with JavaScript (ES6) I realized that learning the series of frameworks known as the “MERN” stack would be the next step. Shortly after experimenting with what I had learned, started building some real-world, useful web aplications using the latest available JavaScript frameworks; they make web development fun. Then, I started to spend time learning some other, better structured, coding languages used for back-end development, such as C# on the .NET framework, Python, and a few other ones as well. Currently, I’m available for hiring as a Full-Stack Web Developer, highly skilled in JavaScript and constantly improving in everything else.
