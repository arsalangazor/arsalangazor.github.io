About Me

            My interest in developing software goes back to the first time I saw a computer; I just knew so deeply that I wanted to learn and be involved in everything happening "Behind-the-Screen." After years of education, finally in 2019 I started to teach myself web development with the goal of becoming a front-end web developer. Out of personal interest, I had programmed in Objective-C for the Mac platform before, but I didn't have a good idea of what development for the web entailed. After getting comfortable with JavaScript (ES6) I realized that learning the series of frameworks known as the “MERN” stack would be the next step. Finally started building some real-world, useful web aplications using the latest available JavaScript frameworks; they make web development fun. I also started to spend some time learning other coding languages, mostly used for back-end development, such as C# on the .NET framework, Python, and a few other ones. Currently, I’m available for hire as a Full-Stack Web Developer, highly skilled in JavaScript and constantly improving in everything else.
